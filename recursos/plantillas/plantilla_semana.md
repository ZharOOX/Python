# Plantilla de semana

Cada semana debe mantener:

- `README.md`
- `objetivos.md`
- `teoria_minima.md`
- `conceptos_clave.md`
- `ejemplos/`
- `ejercicios/`
- `pistas/`
- `soluciones/`
- `mini_proyecto/`
- `errores_comunes.md`
- `checklist_dominio.md`
- `evaluacion.md`
- `resumen_semana.md`

## Flujo de ejercicios

El archivo principal para resolver es `ejercicios/ejercicio_XX.py`.

Cada `.py` de ejercicio debe incluir:

- docstring inicial con titulo, objetivo, conceptos, instrucciones, entrada, salida, restricciones, pistas y reto extra
- separador `ZONA DE TRABAJO`
- codigo inicial, `pass`, `TODO` o comentarios segun corresponda

Los `.md` de ejercicios pueden mantenerse como indice o explicacion extendida. Las soluciones y tests siguen separados.
