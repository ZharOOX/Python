# Materiales

## Minimos

- PC con Python 3.11 o superior.
- PyCharm.
- Acceso a terminal.

## Recomendados

- ESP32 DevKit.
- Cable USB de datos.
- LEDs, resistencias de 220 a 330 ohm.
- Pulsadores.
- Protoboard y cables Dupont.
- Sensor DHT22 o BME280/BMP280.
- Pantalla OLED SSD1306 I2C.

## Opcionales

- Rele de 3.3 V compatible o modulo con transistor/optocoplador.
- Servo pequeno con fuente externa.
- Buzzer.
- Multimetro.

## Sin hardware

Todas las semanas de hardware deben tener una version simulada en Python normal.
