# Curso practico: Python, MicroPython, Flet e IoT con ESP32

Este repositorio es un curso autoguiado de 24 semanas para aprender Python desde una base solida y aplicarlo a automatizacion, datos, interfaces, comunicacion serial, MicroPython, ESP32, web movil local e IoT basico.

## Como empezar

1. Lee `COMO_USAR_ESTE_CURSO.md`.
2. Configura Python y PyCharm con `INSTALACION.md`.
3. Revisa la ruta en `RUTA_COMPLETA.md`.
4. Empieza por `00_preparacion/`.
5. Continua con `01_python_basico/README.md`.

## Metodo de estudio

Cada semana separa teoria, ejemplos, ejercicios, pistas, soluciones y tests. El flujo principal de ejercicios ahora ocurre dentro del archivo `.py`: abre `ejercicios/ejercicio_XX.py`, lee el docstring inicial y escribe tu codigo en la `ZONA DE TRABAJO`.

Los archivos `.md` de ejercicios quedan como respaldo o explicacion extendida, pero no son obligatorios para resolver. Las soluciones siguen separadas en `soluciones/` y debes revisarlas solo despues de intentarlo.

## Ejecutar ejemplos

Desde la carpeta del curso:

```bash
python 01_python_basico/ejemplos/ejemplo_01_temperatura_promedio.py
```

## Resolver ejercicios

```bash
python 01_python_basico/ejercicios/ejercicio_01.py
```

Dentro del archivo veras:

- titulo y objetivo
- conceptos que practica
- instrucciones
- entrada y salida esperada
- restricciones
- pistas breves
- `ZONA DE TRABAJO`

## Ejecutar tests

```bash
pytest
```

## Orden recomendado

Sigue las semanas en orden. Las semanas 1 a 4 estan desarrolladas con ejercicios completos, pistas, soluciones y tests. Las semanas 5 a 24 quedan estructuradas y listas para completar progresivamente.
