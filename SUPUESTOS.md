# Supuestos del curso

- Placa principal: ESP32 DevKit compatible con MicroPython.
- Nivel del estudiante: principiante o intermedio bajo.
- Editor principal: PyCharm.
- Python esta instalado o se instalara durante la semana 00.
- Sistema operativo no especificado; se incluyen comandos para Windows, macOS y Linux cuando corresponde.
- Si no hay hardware, se usan simulaciones en Python normal.
- Para relés, motores o cargas externas se requiere fuente adecuada y GND comun cuando corresponda.
