# Evaluacion

## 5 preguntas de comprension
1. Que problema resuelve el tema de esta semana?
2. Que error comun puedes reconocer ahora?
3. Como aplicarias esto a un sensor?
4. Como verificarias que el programa funciona?
5. Que parte todavia necesitas practicar?

## 5 ejercicios cortos
1. Ejecuta un ejemplo y cambia un valor.
2. Provoca un error controlado y explica el mensaje.
3. Crea una variante minima.
4. Explica una salida con tus palabras.
5. Revisa tu codigo con el checklist.

## Reto practico
Crea una version pequena relacionada con Protocolos, comandos y maquinas de estado.
