# Conceptos clave

Tema: Preparacion del entorno

Conceptos que debes reconocer y practicar esta semana:

- PyCharm, terminal, entorno virtual, pip, Git basico, estructura del curso
- Version minima funcional antes de agregar mejoras.
- Lectura cuidadosa de errores.
- Relacion del tema con sensores, automatizacion, ESP32, Flet o IoT.

Preguntas de control:

1. Que problema practico resuelve este concepto?
2. Que dato entra al programa?
3. Que resultado debe salir?
4. Que error comun podria aparecer?
5. Como lo aplicarias a un proyecto pequeno?
