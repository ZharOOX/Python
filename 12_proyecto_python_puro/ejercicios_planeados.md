# Ejercicios planeados

Mini-proyecto definido: Monitor ambiental de PC sin hardware

Distribucion recomendada:

- 5 ejercicios basicos para entender sintaxis y flujo minimo.
- 5 ejercicios intermedios para combinar el tema con datos ambientales.
- 3 ejercicios aplicados a sensores, automatizacion, comunicacion o interfaz.
- 2 retos libres para modificar, corregir o extender codigo.

Reglas de escritura:

- Mantener el enunciado principal dentro de `ejercicio_XX.py` como docstring autocontenido.
- No incluir la solucion dentro del enunciado.
- Crear version simulada cuando el ejercicio dependa de hardware.
- Agregar tests con pytest cuando sea Python puro y tenga sentido.

- Incluir siempre el separador ZONA DE TRABAJO en cada archivo .py de ejercicio.
- Mantener soluciones y tests en carpetas separadas.
