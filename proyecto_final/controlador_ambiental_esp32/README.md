# Proyecto final: controlador ambiental ESP32

Integra Python en PC, MicroPython en ESP32, sensor real o simulado, actuador real o simulado, comunicacion serial, dashboard Flet, interfaz web desde celular, configuracion JSON, modo manual, modo automatico, logs, manejo de errores, documentacion y pruebas.

## Funciones esperadas desde Flet
- Ver temperatura y humedad.
- Ver estado del ESP32.
- Encender/apagar salida.
- Cambiar modo manual/automatico.
- Modificar umbrales.
- Leer mensajes seriales.
- Guardar datos.

## Funciones esperadas desde celular
- Abrir pagina web servida por ESP32.
- Ver estado del sistema.
- Controlar salida.
- Consultar sensor.
- Cambiar modo.
