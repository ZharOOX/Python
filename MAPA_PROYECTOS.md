# Mapa de proyectos

- Semana 1: simulador de temperatura basico. Entrena variables, operaciones y salida.
- Semana 2: alarma ambiental simulada. Entrena condicionales.
- Semana 3: monitor de lecturas. Entrena bucles y acumuladores.
- Semana 4: analizador ambiental. Entrena listas, tuplas y procesamiento.
- Semana 12: proyecto Python puro. Integra archivos, JSON, modulos y pruebas.
- Semana 22: dashboard Flet serial. Integra interfaz y ESP32.
- Semana 24: controlador ambiental ESP32. Integra PC, ESP32, web, serial, JSON, MQTT y logs.
