# Mini-proyecto: Configurador de controlador ambiental con diccionarios y JSON

## Objetivo

Construir una version minima funcional relacionada con la semana, primero simulada y luego ampliable.

## Version minima

- Usar datos fijos o simulados.
- Mostrar una salida clara.
- Mantener el codigo corto y explicable.

## Mejoras propuestas

- Agregar validacion.
- Separar configuracion cuando el tema lo permita.
- Agregar pruebas si es Python puro.
- Crear version real y version simulada si depende de hardware.

## Criterio de cierre

El mini-proyecto queda aprobado cuando puedes ejecutarlo, explicar cada parte y modificar un valor sin romper el programa.
