# Diccionarios, estados y estructuras JSON

Diccionarios, configuraciones, estados y datos con forma de JSON.

Trabaja en este orden: objetivos, teoria minima, ejemplos, ejercicios, pistas si hace falta, soluciones al final, mini-proyecto y evaluacion.

## Recordatorio antes de empezar

Esta semana es el primer bloque pendiente de ampliacion completa. Antes de estudiarla, revisar `RECORDATORIOS.md` y completar los ejercicios reales si todavia aparecen como "pendiente guiado".

## Flujo de ejercicios en PyCharm

Abre `ejercicios/ejercicio_XX.py` como archivo principal. El enunciado esta dentro del docstring inicial y el codigo se escribe en la `ZONA DE TRABAJO`. Los archivos `.md` quedan como respaldo o explicacion extendida; las soluciones siguen separadas en `soluciones/`.
