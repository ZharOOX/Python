# Evaluacion Semana 1

## 5 preguntas de comprension

1. Por que `input()` entrega texto aunque escribas un numero?
2. Cual es la diferencia entre `24.5` y `"24.5"`?
3. Para que sirve un f-string?
4. Que variable cambiarias para modificar un umbral de alerta?
5. Por que conviene probar primero con valores fijos?

## 5 ejercicios cortos

1. Crea variables para temperatura, humedad y voltaje.
2. Muestra un reporte con unidades usando `print()`.
3. Convierte `20 C` a Fahrenheit.
4. Calcula corriente con `voltaje = 5` y `resistencia = 1000`.
5. Pide una temperatura con `input()` y muestra el valor convertido.

## Reto practico

Crea un diagnostico ambiental que muestre:

- nombre del dispositivo
- temperatura
- humedad
- voltaje
- umbral de temperatura
- mensaje de alerta construido con f-string

No uses `if` todavia. Esa parte empieza en Semana 2.
