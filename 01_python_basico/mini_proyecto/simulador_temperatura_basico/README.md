# Mini-proyecto: simulador_temperatura_basico

Objetivo: construir un simulador minimo de temperatura sin hardware real.

## Versiones incluidas

- `codigo_inicial/main.py`: punto de partida simple.
- `codigo_inicial/version_valores_fijos.py`: estructura para completar con valores fijos.
- `codigo_inicial/version_con_input.py`: estructura para completar con entrada de usuario.
- `codigo_resuelto/main.py`: version minima resuelta.
- `codigo_resuelto/version_valores_fijos.py`: solucion con valores fijos.
- `codigo_resuelto/version_con_input.py`: solucion con `input()`.

## Como trabajar

1. Ejecuta primero la version inicial.
2. Completa la version con valores fijos.
3. Completa la version con `input()`.
4. Revisa las soluciones solo despues.
5. Lee `mejoras/RETOS.md` y elige un reto.

## Comandos

```powershell
python 01_python_basico/mini_proyecto/simulador_temperatura_basico/codigo_inicial/main.py
python 01_python_basico/mini_proyecto/simulador_temperatura_basico/codigo_resuelto/version_valores_fijos.py
python 01_python_basico/mini_proyecto/simulador_temperatura_basico/codigo_resuelto/version_con_input.py
```
