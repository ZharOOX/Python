# Errores comunes

- Escribir `true` o `false` en vez de `True` o `False`.
- Sumar texto con numero: `"24.5" + 1` produce error.
- Olvidar convertir `input()` con `float()` o `int()`.
- Usar una variable con nombre distinto: `temperatura` y luego `tempertura`.
- Dividir por cero al calcular corriente con resistencia `0`.
- Formatear mal un f-string: falta la `f` antes de las comillas.
- Usar coma decimal `24,5`; en Python se usa punto decimal `24.5`.
- Ejecutar la solucion antes de intentar el ejercicio.
- Cambiar muchas lineas al mismo tiempo y no saber que rompio el programa.

Cuando aparezca un error, lee la ultima linea del mensaje y revisa el numero de linea indicado.
