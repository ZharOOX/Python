# Objetivos

Al terminar esta semana deberias poder:

- Crear variables con nombres claros para temperatura, humedad, voltaje, corriente, estado y configuracion.
- Distinguir tipos basicos: `int`, `float`, `str` y `bool`.
- Usar `print()` para mostrar reportes simples.
- Usar `input()` y convertir texto a numero con `float()` o `int()`.
- Calcular promedios, conversiones de temperatura y corriente usando operaciones aritmeticas.
- Formatear salidas con f-strings.
- Crear mensajes simples de diagnostico o alerta.
- Reconocer errores comunes de tipos, nombres de variables y conversiones.
- Modificar un programa minimo sin perder legibilidad.
