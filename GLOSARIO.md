# Glosario

- Python: lenguaje de programacion usado en PC, automatizacion, datos e interfaces.
- Script: archivo ejecutable con instrucciones.
- Variable: nombre que guarda un valor.
- Funcion: bloque reutilizable de codigo.
- Modulo: archivo Python que puede importarse.
- JSON: formato de texto para intercambiar datos estructurados.
- CSV: archivo de tabla separado por comas.
- Serial: comunicacion por puerto USB/UART.
- GPIO: pin digital de entrada/salida.
- PWM: tecnica para simular salida analogica con pulsos.
- ADC: conversor analogico a digital.
- I2C: bus de comunicacion para sensores y pantallas.
- UART: comunicacion serial.
- API: interfaz para pedir o enviar datos.
- Servidor web: programa que responde solicitudes HTTP.
- MQTT: protocolo ligero de mensajeria IoT.
- Broker: servidor MQTT que distribuye mensajes.
- Topic: canal de mensajes MQTT.
